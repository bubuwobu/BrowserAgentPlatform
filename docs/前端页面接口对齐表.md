# 前端页面接口对齐表

## 登录
- POST `/api/auth/login`

## Dashboard
- GET `/api/live/summary`

## 指纹模板中心
- GET `/api/config/fingerprints`
- POST `/api/config/fingerprints`

## Proxy 中心
- GET `/api/config/proxies`
- POST `/api/config/proxies`

## Profile 中心
- GET `/api/profiles`
- POST `/api/profiles`
- POST `/api/profiles/{id}/test-open`
- POST `/api/profiles/{id}/takeover`
- POST `/api/profiles/{id}/unlock`

## 模板中心
- GET `/api/templates`
- POST `/api/templates`

## 任务中心
- GET `/api/tasks`
- POST `/api/tasks`
- GET `/api/tasks/runs`
- GET `/api/tasks/runs/{runId}`

## Live
- SignalR `/hubs/live`
