# 运行说明

## 1. 启动 MySQL
创建数据库：
```sql
CREATE DATABASE browser_agent_platform CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

## 2. 启动 API
修改 `BrowserAgentPlatform.Api/appsettings.json` 的连接串后运行：
```bash
dotnet restore
dotnet run --project BrowserAgentPlatform.Api
```

默认 API 地址：
- `http://localhost:5216`

## 3. 启动 Web
```bash
cd BrowserAgentPlatform.Web
npm install
npm run dev
```

默认 Web 地址：
- `http://localhost:5173`

## 4. 启动 Agent
```bash
dotnet restore
dotnet run --project BrowserAgentPlatform.Agent
```

## 5. 完整流程
- 登录 Web
- 新建指纹模板 / Proxy / Profile
- 在编排器创建任务
- Agent 心跳注册
- API 创建 `task_run`
- Agent 拉到任务并执行
- Live 页看到日志和截图
