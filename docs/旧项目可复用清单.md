# 旧项目可复用清单

这版同步参考并优先复用了旧项目的这些方向：

## 已同步的结构思想
- `BrowserAgentPlatform.Api / Agent / Web` 三层拆分
- 任务中心、模板中心、编排器、Live 页 的总体入口
- Agent 轮询拉任务的运行模式
- Mock 站点用于自动化验证的思路

## 已改造替换的核心
- 旧版 `BrowserPoolManager` 的共享 browser 模式
- 内存服务主干
- 分散的控制器依赖

## 保留但未强行塞入主骨架
- AiComments
- Analytics
- Growth
- Approval
- ContentHub
- ScenarioCenter

原因是这一版优先把可运行主链闭环。
