# TikTok Mock Site

一个可直接运行的仿 TikTok 测试站点，适合用于：

- BrowserAgent / Playwright 自动化
- 可视化流程编排调试
- selector 生成与验证
- 登录 / 点赞 / 评论 / 收藏 / 分享 / 关注 / 主页跳转回归测试

## 启动

```bash
npm install
npm start
```

默认地址：

- http://localhost:3001/login
- http://localhost:3001/feed
